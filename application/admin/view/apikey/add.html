API使用说明

重要提示：

1. 创建后将自动生成 AppID 和 AppSecret，请妥善保管

2. AppSecret 只在创建时显示一次，请立即复制保存

3. 所有API请求都需要携带有效的AppID和AppSecret进行身份验证

4. 如果设置了IP白名单，则只有白名单内的IP才能调用API

API接口列表：

接口名称请求方式接口地址说明
获取商品列表GET/api/goods/getGoodsList分页获取商品列表，支持分类和关键词搜索
获取商品详情GET/api/goods/getGoodsDetail根据商品ID获取商品详细信息
创建订单POST/api/goods/createOrder根据商品ID创建购买订单
获取订单状态GET/api/goods/getOrderStatus根据订单号查询订单状态和卡密信息
获取支付二维码POST/api/goods/getPaymentQRCode生成支付宝或微信支付二维码

基础参数说明：

所有API请求都需要包含以下基础参数：

  • app_id：应用ID，创建API秘钥后获得
  • app_secret：应用秘钥，创建API秘钥后获得

响应格式统一为JSON：

{
  "code": 1,
  "msg": "操作成功",
  "data": {},
  "timestamp": 1234567890
}
